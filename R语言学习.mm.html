    • 数据对象
      • 数据对象的类型
        • 向量
          • vector是最简单的对象，向量由一个或多个同类变量组成
          • 可用使用函数c()来创建一个向量
            • x <- c(1,1,2,2,3)
            • y <- c(3,4,4,5,5)
          • 向量组合
            • z <- c(x, y)
          • 向量下标
            • z[4]
        • 矩阵
          • matrix，与向量差不多，不同的是分成了行和列，即行列式
          • 可以使用矩阵函数matrix()来新建一个矩阵变量
            • m <- matrix(c(2, 3, 1, 5), nrow = 2, ncol = 2)
              • 对向量构建成2行2列的矩阵，先按行分，后按列
            • m <- matrix(c(2, 3, 1, 5), nrow = 2)
              • 对向量构建成2行的矩阵，先按行分，后按列
            • m <- matrix(seq(1, 20, 1), nrow = 5, byrow = TRUE)
              • 生成一个矩阵，指定行数，并按行排列
            • m <- matrix(seq(1, 20, 1), nrow = 5, byrow = FALSE)
              • 生成一个矩阵，指定行数，并按列排列
          • 矩阵的任何一行或一列都是一个向量
          • 矩阵的下标
            • m[2, 2]
        • 数据框
          • dataframe，与矩阵差不多，区别在于，不用的列可以是不同类型。相当于Excel表格
          • 可用data.frame()函数来创建数据框变量
            • d <- data.frame(a, b)
            • a <- c(1,2,3,4)
            • b <- seq(5, 8, by = 1)
          • 数据框的结构打印
            • str(d)
          • 数据框的行数
            • nrow(d)
          • 数据框的列数

          • 更改列名称
            • 多列名称更改

            • 一列名称更改
              • names(h)[1] <- "one"

      • + - 14.1.1参数分布的概念与检验思想
        • + - 统计学的分布概念
          • 一个城市每天的死亡人数，服从“泊松Poisson分布”
          • 一个水体样本某种成分的重复测试的到数据集属于正态分布（中心值就是浓度的真值）
          • 对于结局只有“全或无”的事件，服从于二项分布（扔硬币；产品合格与否；击中球与否等等）
          • 观测和实验得到的数据属于某一种分布
            • 汇率的波动
            • 大气污染的数据
            • 产品的合格率
        • + - 统计检验的基本思想
          • 检验类别
            • 假定分布形式，但参数未知，基于样本数据评估/假设的检验方法，称为“参数方法”
            • 不针对具体参数，而是针对分布的类型进行检验，例如：假设总体分布具有正态性，则可基于样本来检验假设是否可接受
          • 意义
            • 是否服从于特定参数的某种已知分布
            • 检验已知数据集服从于某种已知分布（不指定参数）
          • 什么是参数
            • 参数就是描述总体样本分布的特征的数据，比如：中心值，方差等等，总体的分布由参数决定
            • 参数是确定的，但是往往参数又是不可知，而又无法得到全部样本，只能通过部分抽取的样本进行统计得到的数据，叫做统计量
            • 参数估计：根据样本数据的统计量来推断总体样本的过程
            • 对参数提出一定的假设，然后根据已知统计量对提出的假设进行假设检验
          • 思想

            • 检验过程：概率性质的反证

              推断原理：原假设H0, 基于此前提验证观察到的事件，若此事件发生的概率很小（p值小于0.05），说明小概率事件在观察中居然发生了，那就于“实际推断原理”相悖，假设H0成立时错误的，拒绝此原假设；反之，如果这个事件发生的概率很大，那么在一次观察中发生的可能性很大，我们就该接受原假设。

          • 注意事项

            • 小概率事件在一次观察中发生与实际推断原理相矛盾，但矛盾并不是形式逻辑中的绝对矛盾

              原假设和对立假设不对称，不可交换，在假设检验中的地位不同

              小概率事件在一次观察中既有可能发生，也有可能不发生；若将因此拒绝了H0，就犯错了（犯错的概率为P1，拒绝H0为真）；还有一种错误，H0不正确而我们却接受了H0，犯此错误的概率为P2（接受H0为真）。P1与P2的错误可能会发生，但通常我们解决这类问题简化为只针对第一种错误（P1），限制P1犯错的概率大小，而对第二类P2的错误不管。——意思就是：遵循人类的第一感觉，但对第一次的感觉的怀疑程度越大（p值越大），则犯错P1的概率就越低。

      • 14.1.2.随机变量
        • 随机变量，即面对得到的数据
        • 事件
          • 事件类别
            • + - 定性事件（定性概率）：事件结局是有限的，发生概率不确定，无法预测
              • 产品不合格（合格与不合格），属定性
              • 交通事故：轻微，一般，严重事故，属定性
            • 定量事件（定量概率）：
              • 产品不合格具体到某条产品线，即数值化不合格的比例，属定量
              • 定量随机变量，就是量化的观测值，源于数值型观察值或测量值
                • 可求平均值
                • 标准偏方差
                • 估计随机变量可能误差
              • 定量事件，是由于相关因子引起发生的大概率事件
        • （定量随机变量）变量类别
          • 离散型随机变量
            • 概念：一个变量的观察值只能取可数个数值
            • 特点：可能的取值是孤立和分散的
            • 例如：安装信号灯后十字路口交通的变化；选举无效票数
            • 分布：二项分布、泊松分布、负二项分布、几何分布、超几何分布、多项分布
            • 实验：Bernoulli伯努利试验（独立重复试验）
              • 试验E需满足条件
                • （1）每次试验独立进行
                • （2）每次试验只有两种结果：事件A发生或不发生（结果互逆，完全不相容）
              • 试验概率公式
                • P(A) = p, 0 < p <1, 试验E重复n次，就是n重伯努利试验，P为发生A事件的计算，p为发生A事件的概率值
          • 连续型随机变量
            • 概念：一个变量的观测值可以连续取到某一个区间上的任何一个数值
            • 特点：能取到的数值是无限、不可数的
            • 例如：气温、降水量、股票价格、大气污染物浓度等
            • 分布：正态分布、均匀分布、指数分布等
      • + - 14.1.3.二项分布

        • 二项分布的描述：n重伯努利试验中出现x次成功的概率分布

          R中产生随机样本的命令是：binom(x, size, prob, alternative), x是试验成功次数，size是总试验次数，prob是成功的概率, alternative备选（对立）假设方案：真正成功率取值less，greater, two sided(默认)。若计算出来的结果p-value>0.05则接受原假设，否则接受此设定的备选（对立）假设。

          + - 两个例子

          • 扔硬币，扔了60次，其中27次朝上，检验函数为：binom.test(x = 27, n = 60, p = 0.5, alternative = "less")，意思是说：验证扔硬币这面朝上的概率是否符合0.5的概率，假设对立假设less即不符合0.5的概率，然后进行测试，发现p-value=0.2595，且p>0.05（p<0.05的话，则是小概率事件），说明这是大概率事件且发生，无法拒绝原假设，即扔硬币结果符合二项分布。

            假设一种比赛，一个运动员击球，只看成功率高低，M是一个成功率较高的击球手，根据十年前的赛事成绩，他的成功率为0.4，即40%；新的赛季依赖，他一共击球558次，击中了203次，命中率为0.36，那么保持这个水平，本赛季结束后，他的成功率低于0.40的概率是多少？
            binom.test(x = 203, n = 558, p = 0.4, alternative = "less")
            结果： Exact binomial test
            data:  203 and 558
            number of successes = 203, number of trials = 558, p-value = 0.04378
            alternative hypothesis: true probability of success is less than 0.4
            95 percent confidence interval:  0.0000000 0.3986764
            sample estimates: probability of success  0.3637993
            说明：备选（对立）假设是less：发生小于0.4的概率；但经过计算p-value为0.04378, 说明p值小于0.05，即发生这种小概率事件，所以拒绝原假设，接受备选（对立）假设，即接受下来球员的成功率将低于0.4.

      • + - 14.1.4.泊松分布
        • 背景
          • 1837年由“数学家”泊松提出，作为二项分布的近似引入的
          • 若把伯努利试验中发生概率很小的事件称为稀有事件，那么根据泊松定理，n重伯努利试验中稀有事件发生的概率近似服从泊松分布。
        • 概念与举例
          • 泊松分布是二项分布的极限（总实验次数n很大，发生概率非常小）
          • 自然界和现实生活中，常遇到随机时刻出现的某种事件，我们把在随机时刻相继出现的事件形成的序列，称为随机事件流。
          • 随机事件流具有平稳性、无后效性、普通性，则称该事件为泊松事件流（泊松流）
          • 举例说明
            • 细胞培养板里的细菌个数
            • 一个大城市每天因心血管疾病死亡的人数
            • 热线电话每10分钟的呼入数
            • 一本书各章的错别字数
        • 产生的一般条件
          • 平稳性
            • 在任意时间区间内，事件发生k次（k>=0）的概率只依赖于区间长度而与区间端点无关
          • 无后效性
            • 在不重叠的时间段内，事件的发生相互独立
          • 普通性
            • 如果时间区间充分小，事件出现两次或两次以上的概率可忽略不计
        • 泊松分布参数的概率意义
          • 单位时间出现的随机质点的平均个数
        • R执行泊松分布检验
          • piosson.text(x, T=1, r=1, conf.level=0.95, alternative=c("two.sided", "less", "greater"))
            • x: 预期事件发生的次数
            • T: time base for event count. 用于比较的数据（基线数据）
            • r: 假设或已知的发生概率
            • alternative: 选择“two.sided”, “greater” or "less"
            • conf.level: 置信区间
        • 例子说明

          • 某公司为了宣传产品，开设了一条购物热线，投资方期望这条热线电话每10分钟能有15次呼入，开通后计数表明，10分钟内呼入次数为13，那么这个呼入次数达到了投资方的预期值了吗？

            泊松分布检测：possion.test(13, 15), 13是实际计数得到的值，15是预期值

            测试结果：p-value = 0.6991，大于0.05，应接受原假设，即呼入次数是符合预期的

            某个热线电话10分钟内呼入的电话数为13，而另外一热线电话10分钟内呼入的电话数为24，这两个热线电话的受欢迎程度一致否？--》泊松分布测试：poisson.test(13, 24, alternative = "less")
            测试结果：p-value = 0.01072
            ## 要与第二个热线电话打呼入数持平，临界值（呼入电话数）最多是多少？
            qpois(0.95,  lambda = 24)
            ### [1]   32
            ## 要与第二个热线电话呼入次数持平，临界值（呼入电话数）最少是多少？
            qpois(0.05,  lambda = 24)
            ### [1] 16
            p小于0.05， 拒绝原假设，两个热线受欢迎程度不一样，如果要和第二条热线一样受欢迎，第一条热线的临界呼入次数最多为32，最少16.

            扩展阅读：利用Pearson卡方检验来检验是否符合正态分布：

            # 现有42个数据，分别表示某一时间段内电话总机呼入的次数，

            #呼入的次数 0  1     2    3    4    5     6

            #出现的频率 7  10   12   8    3    2    0

            #问：某个时间段内呼入次数是否符合Possion分布？

            x <- 0:6

            y <- c(7, 10, 12, 8, 3, 2, 0)

            mean <- mean(rep(x, y))

            q <- ppois(x, mean)

            n <- length(y)

            p <- q

            p[1] <- q[1]

            p[n] <- 1-q[n-1]

            for(i in 2:(n-1)) {

              p[i] <- 1 - q[i - 1]

            }

            chisq.test(y, p = rep(1/length(y), length(y)))

            ## Chi-squared test for given probabilities

            ## data: y

            ## X-squared = 19.667, df = 6, p-value = 0.003174

            p < 0.05, 拒绝原假设, 电话呼入次数不符合泊松分布


      • + - 14.1.5.卡方分布

        • + - 概念

          • 如果一个数据集(包含n个数据)服从于正态分布，那么这个数据集力每个数据的平方服从于自由度为n的卡方分布
          • 卡方分布的形状取决于n

          卡方拟合优度检验用来检验样本是否取决于特定类型卡方分布，执行语句是：chisq.test()

          chisq.test(x, y = NULL, correct = TRUE, p = rep(1/length(x), length(x)), rescale.p = FALSE, simulate.p.value = FALSE, B = 2000)

          • x是数据集，可以是数值型向量、矩阵或者因子
          • y是数值型向量。如果x是数值型矩阵，则y可以忽略；如果x是因子，y必须是同样长度因子
          • correct是逻辑型变量，指示是否需要做连续型校正
          • p是指定的概率分布
          • rescale.p是逻辑型变量，指示是否各项p的和为1
          • simulate.p.value是逻辑型变量，指示p的计算过程是否要进行蒙特卡罗模拟
          • B是整数，蒙特卡罗检验里的重复次数

          + - 例子说明

          • 如果连续掷色子200次，得到如下点数分布：

            1    2     3      4      5      6

            30  21  24   35    42    48

            如果色子是均匀的，那么每一面出现的期望概率应该相等，都是1/6.

            但注意到点数5和6竟然分别出现了42次和48次, 这是纯属巧合，还是色子不均匀?

            myfreq <- c(30, 21, 24, 35, 42, 48)

            myprobs <- c(1,1,1,1,1,1)/6

            chisq.test(myfreq, p = myprobs)

            ##测试结果

            #X-squared = 16.3, df = 5, p-value = 0.006038

            p值远小于0.05, 拒绝原假设，这个色子不均匀。

      • 14.1.6.正态分布
        • 概念
          • 生活学习中最常见的连续型随机变量概率分布，很多随机变量的概率分布都可近似地用正态分布来描述
          • 例子
            • 在生产条件不变的条件下，产品的强力、抗压强度、口径、长度等指标
            • 相同生长环境下同种生物的身长、体重等指标
            • 同一批种子的重量
            • 同一实验测量结果
            • 弹着点沿某一方向的偏差
          • 如果一个量是由许多微小的独立随机因素影响的结果，那么就可认为这个量具有正态分布
          • 检验
            • 原假设为H0: 数据集符合正态分布
            • R计算出统计量W，其最大值是1. W越接近1，表示样本越接近正态分布。
            • 若p-value小于显著性a(0, 05), 则拒绝原假设H0，该分布不属于正态分布
        • R实现正态检验函数：shapiro.test(x)

          • par(mfcol = c(1,2), ps = 6.5)

            hist(lh, breaks = 20) #直方图

            qqnorm(lh, pch=1)

            shapiro.test(lh)

        • R中还有ks.test()函数用于检验分布
          • 执行的是Kolmogorov-Smirnov连续分布检验，可以用于包括正态分布的多种连续型概率分布的检验
          • 检验的原理：以样本数据的累计频数分布与特定理论分布比较，若两者间的差距很小，则推送该样本取自某特定分布族
          • 执行结果给出一个统计量D，D是假设的分布累计频数与受检样本之间累计频数分布概率之间的差异，记住：D值越小，越接近0，表示样本数据越接近预先建设的分布

      • 14.1.9.韦伯分布
        • 又称韦氏分布或威布尔分布，是可靠性分析和寿命检验的理论基础。
        • 通常用在故障分析领域（field of failure analysis)中，可以模拟故障率随时间的变化分布
        • 故障率为三种情况：
          • 一直为常数，a=1，提示故障可能在随机事件中发生
          • 一直减少，a<1，提示infant motality先天不足
          • 一直增加，a>1，提示wear out随着时间的持续，故障可能性增加
        • 检验方法Kolmogorov-Smirnov：

          • set.seed(1)

            mydata <- rweibull(8500, 1)

            ks.test(mydata, "pweibull", 1)

            ##One-sample Kolmogorov-Smirnov test

            ##data: mydata

            ##D = 0096477, p-value = 0.4074

            ## alternative hypothesis: two-sided

            • D值很小，p值>0.05, 不能拒绝原假设，该样本服从韦伯分布1型

            ks.test(mydata, "pweibull", 2)

            D = 0.18703, p-value = 2.2e-16

            • 显然不符合2型了
        • Kolmogorov-Smirnov测试检验工具
          • 常用于连续型概率分布检验，不能用离散型概率分布
          • 格式：ks.test(x, y, ..., alternative = c("two.sided", "less", "greater"), exact = NULL)
            • x: 数据集
            • y: 假设的连续型概率分布
            • ...：1型，2型，也可没有
            • alternnative: 备择假设
            • exact: 是否需计算出精确的p值

      • 2.计算均值或方差
        • 1. 单正态总体的体验
          • 存在某一真值，检验某一组数据是否可接受此真值
            • t.test(c[], mu = 316.45)
        • + - 2. 双正态总体的体验
          • 两组数据比较，通常比较平均值，但两组平均值是否存在显著差别？
            • t.test(myco2.t1, myco2.t2)
        • 3. 配对t检验
          • 比较两个数据集时，若彼此独立，可用普通t检验，R的默认设置时paired=FALSE
            • t.test(extra ~ group, data = sleep)
          • 比较两个数据集时，若彼此不独立（有共同的属性），有千丝万缕的联系，则不能用独立样本t检验，需在执行语句设置一个配对参数paired=TRUE
            • t.test(extra ~ group, data = sleep, paired = TRUE)
          • 多组样本数据的配对t检验

            • pairwise.t.test(x, g, p.adjust.method = p.adjust.methods, pool.sd = !paired, paired = FALSE, alternative = c("two.sided", "less", "greater"), ...)

              • x为数据型变量
              • g为一个指定分组的因子型变量
              • pool.sd指定是否计算所有组的统一标准差
        • 4.方差
          • 方差的意义：总体样本的离散程度，即一个数据集里，所有数据的分散程度，分散程度也就是观测值的波动
          • 引起波动的原因
            • 一类是不可控的随机因素干扰或者观测误差
            • 一类是因为不同的处理方式或者实验设计引起的可控波动
          • 方差分析的目的
            • 将数据的总波动分解为上述两类，并且作出数量分析，以比较这两类波动所占的比例，从而进一步分析数据或者改善实验设计
          • 方差的本质
            • 一种均值检验，对方差来源解析，推测某一个或多个因素下各水平的因变量均值是否明显差异，从而判断哪些因素有显著影响
          • 方差分析
            • 单因素方差分析
              • oneway.test(x ~ group, var.equal = T)
                • oneway.test(weight ~ feed, data = chickwts, var.equal = T)
            • 不考虑交互作用的双因素方差分析
            • 考虑交互作用的双因素方法分析
        • 5.非参数假设检验
          • 不依赖某一专门的总体分布的统计分布，称为：不需要分布的统计方法
          • 总体分布不确定，即与参数无关，这样的检验通常将数据转换为秩（Rank）来进行分析
          • 秩，就是数据按照升幂排列后，每个观测值的位置
          • 利用数据的秩来分析，就避免了不知道数据分布的困扰，这就是大多数非参数检验的有点
          • 如何利用秩检验来比较：两个样本数据是否来自同一总体？是否服从同样的分布？是否有同样的中位数？
          • 常用是Wilcoxon检验，也就是不依赖分布的t检验
            • 单样本检验
          • 检验前：往往通过图示得到一个初步的了解
            • 直方图histogram
              • 频数用矩形块标绘，通过直方图推测分布的密度函数对比，得到直观印象
            • 茎叶图Stem-and-leaf Diagram
              • 将数据集中的数据按位数进行比较，把数的大小基本不变或变化不大的位作为主干（茎），把变化大的位的数作为分支（叶），列在主干的后面，这样每个主干后面几个数就一目了然了
            • Q-Q图
              • 仅用于检验正态性

      • 相关性分析

        • 数据分析时，往往需要知道两个变量是否相关

          相关系数可以用来平衡两个随机变量之间的线性相关程度，取值范围是-1到1

          Pearson相关系数

          • 通常用来衡量相关关系的统计量是Pearson相关系数，也正是Excel软件中用CORREL函数计算出来的结果
          • 因为Pearson相关系数是基于正态分布计算的，因此对于服从正态分布的变量来说度量效果更好
          • cor(iris.data)

          Spearman相关系数

          • 是一个非参数统计量，它不对数据的分布情况作出假设
          • 数据不服从正态分布，甚至不知道是否服从于某一种已知的分布

          Kendall T

          • 它比较的是两个随机变量的秩，而不是比较随机变量的数值
          • 对变量的分布情况没有要求

          cor.test(x, y, alternative = c("two.sided", "less", "greater"), method = c("pearson", "kendall", "spearman"), exact = NULL, conf.level = 0.95, continuity = FALSE, ... )

          • x, y 需要检测的数据值
          • alternative：备择假设
          • method：三种统计量，根据数据情况选一种
          • exact：是否需要精确计算出p值
          • conf.level: 置信区间，只对pearson相关系数有效（要求xy各包含至少四个数据）
          • continuity：是一个逻辑变量，如果选择TRUE，则对Kendalls检验的tau值和Spearman's检验的rho值进行连续性矫正
      • 协方差
        • 是与相关系数紧密联系的一个统计量
        • 协方差的定义正好是Pearson相关系数公式中的分子项
        • 协方差就是两个数据集的方差，体现两个数据集的变化趋势关系
          • 如果协方差未正值，则表明两个数据集变化趋势一致
          • 负值的话，则表示变化趋势相反
          • 0表示两个数据集变化趋势不相关
        • cov.test(x, y = NULL, use = "everything", method = c("pearson", "kendall", "spearman"))
